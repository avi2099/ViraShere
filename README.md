# ViraSphere 🌍

**"One place for the world's most viral moments"**

A beautiful, fully responsive single-page web application that aggregates the world's most viral content — news, social buzz, videos, memes — all in one place.

![Dark Mode](https://img.shields.io/badge/Theme-Dark%20Mode-blueviolet) ![Responsive](https://img.shields.io/badge/Responsive-Mobile%20First-cyan) ![No Dependencies](https://img.shields.io/badge/Dependencies-None-green)

---

## ✨ Features

- **6 Content Tabs** — Home, News, Social Buzz, Viral Videos, Memes, Community
- **Glassmorphism UI** — Frosted glass cards with neon purple, cyan & pink accents
- **Dark / Light Mode** — Toggle with persistent preference
- **Global Search** — Real-time filtering across all tabs
- **Smart Filters** — Region (6 zones), Time (24h / week / all), Sort (hot / latest / top)
- **Upvote System** — Live counters persisted in localStorage
- **Share Buttons** — X/Twitter, WhatsApp, Copy link with toast notifications
- **Community Contributions** — Submit viral content via modal form, saved to localStorage
- **Video Player** — YouTube iframe modal with autoplay
- **Refresh Viral Data** — Shuffles content and injects new mock items
- **Load More** — Infinite-scroll style pagination
- **Fully Responsive** — Mobile bottom tab bar, hamburger menu, horizontal filter pills

## 🚀 Getting Started

No build tools, no dependencies — just one HTML file.

### Option 1: Open Locally
```
Double-click index.html
```

### Option 2: Local Server
```bash
npx serve .
```

### Option 3: Deploy
| Platform | How |
|----------|-----|
| **GitHub Pages** | Settings → Pages → Deploy from `main` |
| **Netlify** | Drag & drop at [app.netlify.com/drop](https://app.netlify.com/drop) |
| **Vercel** | `npx vercel --prod` |
